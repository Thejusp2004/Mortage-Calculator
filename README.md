# 🏦 Mortgage Repayment Calculator

A simple and interactive mortgage repayment calculator built using Streamlit. The application allows users to calculate monthly repayments, view the complete payment schedule, and analyze loan trends through visualizations.

---

## ✨ Features

- Calculate monthly mortgage repayments (EMI)
- Display total repayment amount
- Display total interest paid
- Detailed payment schedule table
- Remaining loan balance analysis graph
- Interactive and user-friendly Streamlit interface

---

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/Thejusp2004/Mortgage-Calculator.git
```

### 2. Move into the project folder

```bash
cd Mortgage-Calculator
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run Home.py
```

---

## 📊 Application Pages

### 🏠 Home
- Input home value, deposit, interest rate, and loan term.
- Displays monthly payment, total repayment, and total interest.

### 📋 Payment Schedule
- Shows month-by-month loan repayment details.
- Includes principal paid, interest paid, and remaining balance.

### 📈 Analytics
- Displays graphical analysis of remaining loan balance over time.
